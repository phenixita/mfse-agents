# The Software Factory Method - Michele Ferracin Software Engineering

This repository contains practical resources for The Software Factory Method: AI agents, reusable skills, and a few utility scripts.

Reference book (Italian): [Il Metodo Software Factory](https://www.amazon.it/dp/B0G6MD5J9N)

## Work In Progress

This repository will always be a work in progress.
It is built from real consulting projects and from years of scripts and tools that were originally scattered across different drives and environments.
The content is continuously reviewed, cleaned up, and expanded over time.

## What You Get

- Coding agent skills, custom agents and team of agents
- Scripts for Azure DevOps
- Templates to improve requirements collection and writing of user stories, features, epics
- Kanban best practices and cheat-sheet
- ... and more!
